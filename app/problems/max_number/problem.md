# Max Number

## Problem Statement

Given an array of integers, return the maximum value in the array.

## Input

- An array of integers, `arr`.

## Output

- The maximum value in the array.

## Example

### Example 1

**Input:**
[3, -2, 1]

**Output:**
3

## Constraints

- `1 <= arr.length <= 10^4`
- `-10^4 <= arr[i] <= 10^4` for each `0 <= i < arr.length`
